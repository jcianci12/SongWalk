User story:
Jon - who is a music lover and a bit of a nerd, spins up his own songshare docker stack (or runs the song share exe) this sets up a songshare folder. now jon can hit a url, and he can drag and drop music to the folder. he can share that songshare url with other users (its a uuid that cant be guessed) and other users can also upload and edit songs in that folder.

Jon is pleased that the user interface is familiar and productive. It looks alot like windows media player legacy.

Jon is able to set up a nginx server that routes traffic to his songshare stack.
